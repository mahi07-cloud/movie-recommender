# 🎬 MovieEngine AI: Machine Learning Recommender

A professional-grade movie recommendation engine built with **Collaborative Filtering** and **Streamlit**.

## 🚀 Features
* **ML Engine:** Uses Item-Item Collaborative Filtering and Cosine Similarity.
* **Real-time UX:** Interactive dashboard for instant movie suggestions.
* **API Integration:** Dynamic movie poster fetching via TMDB API.
* **Optimized Performance:** Implements Streamlit caching for sub-second response times.

## 🛠️ Tech Stack
* **Language:** Python 3.11
* **Data Science:** Pandas, Scikit-learn
* **Interface:** Streamlit
* **API:** TMDB (The Movie Database)

## 📦 Installation
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/movie-recommender.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
